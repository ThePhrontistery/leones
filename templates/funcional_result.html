{# Fragmento HTML para mostrar el documento funcional generado en el panel Markdown #}

Documento funcional generado

Pasos realizados:

  1. Se leen los documentos cargados en el panel de carga de documentos.
  2. Se extrae el texto de cada documento (.txt, .md, .pdf, .docx).
  3. Se lee la plantilla funcional y se extraen las secciones principales.
  4. Se construye un prompt para la IA con la plantilla y los textos de los documentos.
  5. Se realiza una llamada a Azure OpenAI para generar el documento funcional.
  6. Se muestra el documento funcional generado en este panel.
  7. El índice del documento funcional se muestra en el panel Árbol de Contenidos.
